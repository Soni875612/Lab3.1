## 📄 Lab Activity 3.1 — Statistical Analysis of Wheat and Rice Production
🎯 Objective
To perform a statistical analysis of wheat and rice production across various Indian states using NumPy and Pandas, by computing and interpreting Mean, Median, and Mode.

🛠️ Requirements

Python — Jupyter Notebook or Google Colab
Libraries: pandas, numpy, scipy, matplotlib


📚 Prerequisites

Basic Python programming knowledge
Familiarity with NumPy and Pandas for data manipulation
Understanding of descriptive statistics (Mean, Median, Mode)
Knowledge of CSV data handling and basic data visualization

📝 Conclusion

Mean gives the overall average production level across states
Median reflects the central tendency while reducing the effect of outliers
Mode highlights the most frequently occurring production value

This analysis helps policymakers and agricultural planners assess crop production trends and make data-driven decisions to improve agricultural output.
